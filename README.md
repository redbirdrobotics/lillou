# Lillou Project Repositories

This repository contains both of the codebases used on the 2026 IEEE SoutheastCon robot. They are attached as submodules. By clicking them it'll take you to their respective repos. If using locally, be sure you run:

```sh
git submodule update --init --recursive
```

or initialize them in the desktop app to make sure they are pulled down and up to date. From there, each one details how exactly to upload the code and run it.
